================================================================================
TOTAL STRUCTURE: 5 Main Layers + Shared Layer + Configuration + Tests + Docs
================================================================================

LAYER RESPONSIBILITIES:
─────────────────────────────────────────────────────────────────────────────

1. PRESENTATION LAYER (presentation/)
   └─ Streamlit user interface
      • Pages for application, status, analytics
      • Reusable components (forms, cards, charts)
      • Session state and API client

2. MICROSERVICES LAYER (microservices/)
   └─ FastAPI REST API backend
      • RESTful endpoints for loans, status, analytics
      • Pydantic schemas for validation
      • SQLAlchemy ORM models
      • Middleware for auth, logging, error handling
      • Dependency injection

3. ORCHESTRATION LAYER (orchestration/)
   └─ LangGraph workflow management
      • Graph definition and state management
      • Workflow implementations
      • Conditional routing
      • State preservation

4. AGENT LAYER (agents/)
   └─ Four specialized AI agents
      • Agent 1: Loan Processor (extracts & validates)
      • Agent 2: Credit Analyzer (analyzes credit)
      • Agent 3: Risk Assessor (evaluates risk)
      • Agent 4: Decision Maker (makes final decision)

5. MCP INTEGRATION LAYER (mcp/)
   └─ Model Context Protocol for tools & resources
      • Tools: Database, APIs, Calculations, Documents
      • Resources: Credit rules, Compliance, Reference data
      • Request handling and execution

6. SHARED LAYER (shared/)
   └─ Cross-cutting utilities
      • Configuration management
      • Database utilities
      • Custom exceptions
      • Validators, formatters, helpers

7. CONFIGURATION (config/)
   └─ Environment-based settings
      • Database, logging, MCP, agents config

8. TESTS (tests/)
   └─ Comprehensive test coverage
      • Unit tests
      • Integration tests
      • Test fixtures

9. DOCUMENTATION (docs/)
   └─ Architecture & implementation guides
      • Architecture details
      • API documentation
      • Agent specifications
      • Workflow descriptions
      • MCP integration guide
      • Deployment instructions

================================================================================
DATA FLOW:
─────────────────────────────────────────────────────────────────────────────

User Input (Streamlit)
    ↓ HTTP POST
FastAPI (Validation, Database Save)
    ↓ Invoke
LangGraph (State Setup, Routing)
    ↓ Call Agent Nodes
Agent 1 (Loan Processor) → Agent 2 (Credit Analyzer) → Agent 3 (Risk Assessor) → Agent 4 (Decision Maker)
    ↓ Use Tools/Resources via MCP
MCP Layer (Database Tools, APIs, Rules, Calculations)
    ↓ Return Results
Back through Agents & Orchestration
    ↓ Save Decision
FastAPI (Update Database)
    ↓ JSON Response
Streamlit (Display Results)

================================================================================
GETTING STARTED:
─────────────────────────────────────────────────────────────────────────────

1. Generate folder structure:
   $ python create_structure.py

2. Install dependencies:
   $ pip install -r requirements.txt

3. Configure environment:
   $ cp .env.example .env
   # Edit .env and add your API keys

4. Run API server:
   $ uvicorn microservices.main:app --reload

5. Run Streamlit UI (in another terminal):
   $ streamlit run presentation/app.py

6. Access:
   • UI: http://localhost:8501
   • API: http://localhost:8000
   • API Docs: http://localhost:8000/docs
