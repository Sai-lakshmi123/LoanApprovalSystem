💻 CLIENT TYPES
════════════════════════════════════════════════════════════════════════════════

SyncMCPClient (Recommended for most use cases)
├─ Synchronous API
├─ Easy to use in regular functions
├─ Best for FastAPI, agents, scripts
└─ Example:
    client = SyncMCPClient()
    result = client.get_complete_analysis("APP001")

MCPClient (For async/await patterns)
├─ Asynchronous API
├─ Use with async functions
├─ Best for high-throughput services
└─ Example:
    client = MCPClient()
    result = await client.get_complete_analysis("APP001")
