🎯 WHAT WAS BUILT
════════════════════════════════════════════════════════════════════════════════

An intelligent loan decision synthesizer powered by Claude Sonnet 4.6 and built
with the Anthropic Agent SDK. Agent3 synthesizes final lending decisions by
integrating outputs from Agent1 (Application Profile) and Agent2 (Financial
Risk Analysis).

Returns:
  ✅ Decision Classification (APPROVE/CONDITIONAL/REVIEW/REJECT)
  ✅ Risk Score (0-5 scale) synthesized from all factors
  ✅ Confidence Level (Very High to Very Low) with percentage
  ✅ Key Decision Factors (7 dimensions evaluated)
  ✅ Risk Summary (Strengths, Concerns, Mitigating Factors, Critical)
  ✅ Metrics Integration (All inputs synthesized)
  ✅ Recommendation Rationale (Executive summary)
  ✅ Compliance Notes & Audit Trail
  ✅ Structured JSON Output

🔧 4 TOOLS AVAILABLE
════════════════════════════════════════════════════════════════════════════════

Tool 1: synthesize_loan_decision
  Purpose: Synthesize final decision from combined analyses
  Input: applicant_id, applicant_data, application_analysis, risk_assessment, strategy
  Returns: Decision with classification, risk score, confidence, reasoning

Tool 2: evaluate_multiple_scenarios
  Purpose: Evaluate decision outcomes across loan scenarios
  Scenarios: Different loan amounts (Conservative, Moderate, Aggressive)
  Returns: Decision outcomes for each scenario

Tool 3: explain_decision
  Purpose: Get detailed explanation of decision
  Details: Key factors, reasoning, compliance notes
  Returns: Comprehensive decision explanation

Tool 4: compare_applicants ⭐ (Comparative Analysis)
  Purpose: Compare decisions across multiple applicants
  Metrics: Relative risk profiles, comparative recommendations
  Returns: Comparative analysis across applicants


📊 SYSTEM PROMPT
════════════════════════════════════════════════════════════════════════════════

The agent operates with a 150+ line system prompt that instructs Claude to:

ROLE:
  Expert loan decision analyst responsible for final lending decisions

ANALYSIS FRAMEWORK:
  1. Decision Classification (4 types)
     APPROVE: Risk < 1.5, strong profile, clear repayment
     CONDITIONAL_APPROVE: Risk 1.5-2.5, minor concerns, conditions required
     REVIEW: Risk 2.5-3.5, mixed signals, senior judgment needed
     REJECT: Risk > 3.5, critical violations, insufficient mitigation

  2. Risk Score (0-5 scale)
     Synthesizes risk from:
     - Agent1 outputs (income, employment, credit, completeness)
     - Agent2 outputs (DTI, credit risk, loan risk, anomalies, aggregate)
     - Compensating factors analysis

  3. Confidence Level (0-100%)
     Very High (85-100%): Complete data, strong agreement
     High (70-84%): Good data quality, minor issues
     Moderate (55-69%): Acceptable data, some concerns
     Low (40-54%): Limited data or conflicts
     Very Low (<40%): Incomplete data, major conflicts

  4. Key Decision Factors (7 Dimensions)
     Income Stability & Employment Security
     Debt Management Capacity (DTI)
     Credit History & Payment Discipline
     Loan Amount Appropriateness (LTI/LTV)
     Collateral Protection
     Anomalies & Red Flags
     Compensating Factors

OUTPUT FORMAT:
  Valid JSON with exact required fields
  - applicant_id
  - analysis_timestamp
  - decision (classification, risk_score, confidence, reasoning, conditions)
  - key_factors (category, impact, description, score)
  - risk_summary (strengths, concerns, mitigating_factors, critical_factors)
  - metrics_integration (all integrated metrics)
  - recommendation_rationale
  - next_steps
  - compliance_notes (fair_lending_compliant, documentation_complete, audit_trail_created)

💡 USAGE EXAMPLE
════════════════════════════════════════════════════════════════════════════════

from agents.loan_decision_agent import LoanDecisionAgent

agent = LoanDecisionAgent()

result = agent.make_decision(
    applicant_id="APP001",
    applicant_data={
        "name": "John Strong",
        "age": 45,
        "annual_income": 150000,
        "existing_monthly_debt": 1000,
        "credit_score": 760,
        "delinquencies": 0,
        "years_at_current_job": 8
    },
    applicant_analysis={
        "income_stability": {"score": 85},
        "employment_risk": {"risk_score": 15},
        "credit_history": {"credit_score": 760},
        "application_completeness": {"completeness_percentage": 100}
    },
    risk_analysis={
        "dti_analysis": {"dti_with_new_loan_percentage": 35.5},
        "credit_risk": {"adjusted_risk_level": "Excellent"},
        "loan_amount_risk": {"overall_loan_risk": "Very Low"},
        "anomaly_detection": {"anomaly_count": 0},
        "aggregate_risk_assessment": {"overall_risk_score": 1.8}
    },
    strategy="Balanced"
)

if result["status"] == "success":
    decision = result["decision"]["decision"]
    print(f"Classification: {decision['classification']}")
    print(f"Risk Score: {decision['risk_score']}/5")
    print(f"Confidence: {decision['confidence_percentage']}%")
    print(f"Reasoning: {decision['reasoning']}")

🔴 7 KEY DECISION FACTORS
════════════════════════════════════════════════════════════════════════════════

1. Income Stability & Employment Security
   Score Source: Agent1 Income Stability Score
   Scale: Very Unstable → Very Stable (0-100)
   Impact: Foundation for repayment ability

2. Debt Management Capacity (DTI)
   Score Source: Agent2 DTI Analysis
   Scale: Very High Risk (>50%) → Very Low (<20%)
   Impact: Ability to service new loan

3. Credit History & Payment Discipline
   Score Source: Agent1 & Agent2 Credit Data
   Scale: Very Poor (<600) → Excellent (750+)
   Impact: Demonstrated payment reliability

4. Loan Amount Appropriateness
   Score Source: Agent2 LTI & LTV Analysis
   Scale: Very High Risk → Very Low Risk
   Impact: Loan-to-income and loan-to-value ratios

5. Collateral Protection
   Score Source: Agent2 LTV Percentage
   Scale: >95% → <60%
   Impact: Lender recovery protection

6. Anomalies & Red Flags
   Score Source: Agent2 Anomaly Detection
   Count: 0-9 types
   Severity: Critical, High, Medium, Low
   Impact: Unusual patterns and concerns

7. Compensating Factors
   Examples: High income offsetting DTI, strong credit
   Impact: Mitigates specific concerns
   Assessment: Contextual evaluation

🚨 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

Issue: Connection Refused
  Cause: DecisionSynthesis server not running
  Solution: python mcp/decisionsynthesis/server.py

Issue: Invalid API Key
  Cause: ANTHROPIC_API_KEY not set
  Solution: export ANTHROPIC_API_KEY=sk-...

Issue: JSON Parse Error
  Cause: Claude output format invalid
  Solution: Check response, may need prompt adjustment

Issue: Tool Not Found
  Cause: DecisionSynthesis missing tool
  Solution: Verify all 4 tools available in server

Issue: Timeout
  Cause: Slow server or network
  Solution: Increase timeout: DecisionSynthesisClient(timeout=60)

Issue: Decision takes too long
  Cause: Tool execution delay
  Solution: Check DecisionSynthesis server logs
